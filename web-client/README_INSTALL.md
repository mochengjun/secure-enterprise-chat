# Web客户端安装包说明

## 📦 安装包信息

- **文件**: `secure-chat-web-Static-1.0.0.zip`
- **位置**: `installer/windows/output/`
- **大小**: 347 KB
- **版本**: 1.0.0
- **更新日期**: 2026-02-23

## 🚀 快速开始

### 方式一：直接部署到Web服务器

1. **解压安装包**
   ```
   解压 secure-chat-web-Static-1.0.0.zip
   ```

2. **部署到Web服务器**
   - 将解压后的文件复制到Web服务器的根目录
   - 访问您的网站URL即可使用

3. **示例部署位置**
   - Nginx: `/var/www/html/`
   - Apache: `/var/www/html/`
   - IIS: `C:\inetpub\wwwroot\`

### 方式二：本地测试

1. 解压安装包
2. 运行快速启动脚本（需要Python）
   ```
   start-dev.bat
   ```
3. 在浏览器中访问: `http://localhost:8080`

### 方式三：使用Docker

```bash
# 在 web-client 目录下
docker build -t secure-chat-web:1.0.0 .
docker run -p 80:80 secure-chat-web:1.0.0
```

## 🐛 已知问题及解决方案

### 问题1: 粘贴图片不工作
**原因**: 浏览器不支持Clipboard API或用户未与页面交互
**解决**: 
- 使用最新版本浏览器
- 确保用户已与页面有过交互（点击、输入等）

### 问题2: 消息提示音不响
**原因**: 浏览器音频权限或AudioContext初始化问题
**解决**: 
- 检查浏览器音频权限设置
- 检查系统音量
- 在页面任意位置点击以激活音频上下文

### 问题3: WebSocket连接失败
**原因**: WS_URL配置错误或CORS策略问题
**解决**:
- 检查 .env.production 配置
- 确保后端支持WebSocket
- 配置正确的CORS策略

### 问题4: API请求失败
**原因**: API_BASE_URL配置错误或网络问题
**解决**:
- 检查 .env.production 配置
- 验证后端服务是否运行
- 检查网络连接
